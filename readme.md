# Aetheris White Noise Plugin (for NVDA)

The name **Aetheris** comes from the Latin word *aether*, meaning “sky,” “upper air,” or “something light and ethereal.”  
This name reflects the purpose of the plugin: to create a gentle, atmospheric sound environment—like air surrounding you—helping you stay focused, relaxed, or comfortable while using NVDA.

Aetheris is a white noise and ambient sound plugin designed for NVDA.  
It allows you to play multiple audio tracks at the same time, adjust each track’s volume independently, enable random playback, and manage everything through a simple and accessible settings interface.

---

## ✨ Features

### 🎵 Multi‑track Playback
- Play multiple audio files simultaneously  
- Each track has its own volume control  
- Supports loop playback or random playback  

### 🎲 Random Playback Mode
- Plays selected tracks at random intervals  
- Ideal for natural ambient sounds such as birdsong, wind, or forest ambience  

### 🔄 Automatic Playback Restore
- Optionally restore the previous playback state when NVDA starts  

### 🖥 Simple and Accessible Settings Interface
- Choose an audio directory  
- Browse categories and audio files  
- Enable or disable individual tracks  
- Adjust volume and random playback mode  
- View a list of currently active tracks  

---

## ⌨ Keyboard Shortcuts

| Shortcut | Function |
|----------|----------|
| **NVDA+Shift+W** | Open the Aetheris settings dialog |
| **NVDA+Shift+P** | Toggle white noise playback on or off |

You can customize these shortcuts in NVDA’s “Input Gestures” dialog.

---

## ⚙ How to Use

### 1. Open the Settings Dialog
Press **NVDA+Shift+W** to open the Aetheris settings window.

### 2. Choose an Audio Directory
Click the “Browse” button and select a folder containing your audio files.  
Subfolders will automatically appear as categories.

### 3. Select a Category and Audio File
- Choose a folder under “Category”  
- Select an audio file from the list  

### 4. Configure the Track
- **Enable Track**: Start playing the selected audio  
- **Random Playback**: Play the track at random intervals  
- **Volume Slider**: Adjust the track’s volume  

### 5. Apply Your Settings
Click **Apply** or **OK** to save your changes and activate them immediately.

---

## 📂 Configuration Saving

The plugin automatically saves:

- The selected audio directory  
- Enabled tracks  
- Volume and random playback settings for each track  

When you:

- Delete audio files  
- Change the audio directory  

Aetheris will automatically clean up invalid entries to keep your configuration tidy and free of leftover data.

---

## 🧹 Stopping Playback

Press **NVDA+Shift+P** to stop all tracks.  
After stopping:

- All players are closed  
- All random timers are cancelled  
- No background processes remain  

---

## ℹ Notes

- Supported audio formats: **WAV, MP3, OGG, FLAC**  
- If the selected directory contains no categories or audio files, the lists will appear empty  
- Random playback works best with natural ambient sounds such as birds, wind, or water
